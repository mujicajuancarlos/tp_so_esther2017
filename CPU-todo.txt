CPU

1.Iniciar Cpu A KERNEL(handshake)
	
	-> Espera PCB del kernel(package)
	-> Ejecuta PCB  (rafagas de ciclo de intrucción)
	-> Incrementa ProgramCounter
	-> Recibe de MEMORIA sentencia a ejecutar( utilizando IndiceDeCodigo )

2.Parsear Sentencias
	-> ejecutar operaciones requeridas
	-> send valores para actualizar en MEMORIA
	-> update ProgramCounter en PCB

3. Informa al kernel FIN DE EJECUCIÓN
	-> cuando ejecuta su ultima sentencia

4. Varias conexiones CPU-KERNEL
	-> SIGUSR1 (notificar CPU que deberá desconectarse concluida la ejecucción)

5.Finalizar programa ANSISOP actual
	-> Recibe excepción (KERNEL O MEMORIA)
	-> Mensaje de cual fue el error

6. Entender y ejecutar ANSISOP
	-> Agregar elementos a indiceDeStack (args,vars,retPos,retVar)
	-> AnSISOP_funciones functions

7.Manejar STACK

8. CPU-MEMORIA
	-> Recibe los package de memoria
	-> Desempaquetar Package
	-> Empaquetar
	-> Enviar package a memoria

9. Implementar syscall  (por medio del FileSystem)
	-> leer Archivos
	-> escribir Archivos
	-> Borrar Archivos desde ANSISOP

10.CPUs en forma paralela.
	-> Implementar operaciones que requieren acceso al heap de los programas
